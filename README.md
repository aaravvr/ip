# Goofy

> "A-hyuck! I may be clumsy, but I'll never forget your tasks!"

Goofy is a Disney-themed command-line task manager that runs in your terminal. *(Note: This project is a customized fork of the greenfield Java project, Duke).*

## Documentation

[Read the Full User Guide](https://aaravvr.github.io/ip/) for installation steps, the command cheat sheet, and more.

## Quick Start

Download the latest `goofy.jar` from [Releases](https://github.com/aaravvr/ip/releases) and run:
```
java -jar goofy.jar
```
